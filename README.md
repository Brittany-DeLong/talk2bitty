## 🎯 About

**talk2bitty** is a modern web platform that enables clients to:
- 📝 Sign up and authenticate securely
- 💳 Book and pay for consulting/coaching sessions
- 💬 Engage with a community wall
- 📱 Connect via text-based sessions at **(614) 321-9435** (Google Voice)

Perfect for coaches, consultants, therapists, or any professional offering time-based services.

## ✨ Key Features

### 🔐 **User Authentication**
- Netlify Identity integration for secure sign-in/sign-up
- Anonymous browsing option for first-time visitors
- Protected dashboard for authenticated users
- Seamless logout functionality

### 🎟️ **Session Booking**
- Multiple session duration options (30-min, 60-min)
- Integrated Stripe payment processing
- Real-time booking confirmation
- Contact information provided after payment

### 💬 **Community Wall**
- Public community engagement space
- Anonymous & authenticated posting
- Like/react to posts
- Broadcast messaging capabilities

### 👤 **User Dashboard**
- Personal account management
- Session history and bookings
- Account settings via Netlify Identity
- Quick access to contact information
